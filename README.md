# Frontend Service (BFF)

## Описание

`Frontend Service` — BFF (Backend For Frontend) микросервис, который выступает в роли посредника между пользовательским интерфейсом и внутренними микросервисами банковской системы.

Микросервис **не имеет собственной базы данных** и отвечает за:
- Аутентификацию и управление сессиями пользователей
- Агрегацию данных из различных микросервисов
- Предоставление MVC-страниц для пользовательского интерфейса
- Маршрутизацию запросов к соответствующим сервисам

---

## Архитектура

Сервис взаимодействует со следующими микросервисами:
- **User Service** — управление пользователями и аутентификация
- **Card Service** — управление банковскими картами
- **Document Service** — работа с документами
- **Transfer Service** — выполнение переводов
- **User Restriction Service** — проверка ограничений пользователей

---

## Зависимости

Для работы сервиса необходимы следующие микросервисы (должны быть доступны по сети):
- `user-service`
- `card-service`
- `document-service`
- `transfer-service`
- `user-restriction-service`

---

## Переменные окружения

| Переменная | Описание | Значение по умолчанию |
|------------|----------|---------------------|
| `USER_SERVICE_URL` | URL сервиса пользователей | `http://localhost:8081` |
| `CARD_SERVICE_URL` | URL сервиса карт | `http://localhost:8082` |
| `DOCUMENT_SERVICE_URL` | URL сервиса документов | `http://localhost:8083` |
| `TRANSFER_SERVICE_URL` | URL сервиса переводов | `http://localhost:8084` |
| `USER_RESTRICTION_SERVICE_URL` | URL сервиса ограничений | `http://localhost:8085` |
